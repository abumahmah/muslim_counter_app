
# Muslim Counter App

A Flutter app designed to help Muslims count tasbeeh and track dhikr easily.

## About Me
I’m Mohammed Abubakar, a new GitHub user and Computer Science student.  
This is one of my first projects uploaded here as I learn how to use Git and GitHub.

## Features
- Simple tasbeeh counter
- Tap sound feedback
- Cross‑platform support (Android, iOS, desktop, web)

## Getting Started
Clone the repository and run:
```bash
flutter pub get
flutter run
```
